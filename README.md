# 🥁 Drum Day — Gerador de Inscrições

## Como usar (primeira vez)
1. Instale o Python 3.11+ em https://python.org
2. Clique duas vezes em `setup_venv.bat` — instala tudo automaticamente
3. Para rodar o programa: clique em `run.bat`
4. O navegador abre sozinho em http://127.0.0.1:5000

## Uso diário
- Basta clicar em `run.bat`

## Gerar .EXE (opcional)
- Execute `build_exe.bat`
- O arquivo final estará em `dist/DrumDay.exe`
- Esse .exe pode ser enviado para qualquer PC Windows sem precisar instalar Python!

## Como usar o programa
1. Carregue o template (imagem roxa padrão)
2. O círculo é detectado automaticamente
3. Carregue a foto do participante
4. Ajuste zoom e posição com os sliders
5. Digite o nome e baixe o PNG final
